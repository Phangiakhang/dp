Bài 1: Dãy con tăng dài nhất (LIS)
Khởi tạo dp[i] = 1 ∀ i ∈ [0,n-1]
Công thức quy hoạch:
dp[i] = max(dp[i],dp[j]+1)
j ∈ [0,i-1], a[i] > a[j]

Bài 2: Bài toán cái balo 0-1
Ta có:
+ w[i] là trọng lượng đồ vật thứ i
+ v[i] là giá trị của đồ vật thứ i

Bài 3: Dãy con có tổng bằng S
Khởi tạo dp[i] = false ∀ i ∈ [0,S]
dp[i] = true nếu có thể chọn ra tập con có tổng bằng i
=> Kết quả bài toán là dp[S]
Cơ sở: dp[0] = true
Công thức quy hoạch:
dp[j] = dp[j-a[i]]
i ∈ [0,n-1]
j ∈ [S -> a[i]]

Bài 4: Xâu con đối xứng dài nhất
dp[i][j] là xâu con bắt đầu từ i và kết thúc ở j
Khởi tạo dp[i][i] = true ∀ i ∈ [1,s.size()]
1. Nếu s[i] != s[j] thì dp[i][j] = false
2. Nếu s[i] == s[j] và dp[i+1][j-1] == true => dp[i][j] = true
==> dp[i][j] = dp[i+1][j-1] && s[i] == s[j]

3. Nếu chiều dài của xâu i -> j là 2 thì ta chỉ quan tâm tới s[i] == s[j]
gọi l = j-i+1
==> dp[i][j] = l == 2 && s[i] == s[j]
Vậy công thức tổng quát là:
┌───────────────────────────────────────────────────────────────────────┐
| dp[i][j] = (dp[i+1][j-1] && s[i] == s[j]) || (l == 2 && s[i] == s[j]) |
└───────────────────────────────────────────────────────────────────────┘
